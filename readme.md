# npm-email-cli

> Get the email of an npm user

## Install

```sh
npm install --global npm-email-cli
```

## Usage

```
$ npm-email --help

  Usage
    $ npm-email <username>

  Example
    $ npm-email sindresorhus
    sindresorhus@gmail.com
```
